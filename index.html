📊 MODULE 11 — Point Estimate of Population Proportion & Standard Error COMPUTATION

Teacher's Note: Focus on COMPUTATIONS. Formulas will be given — know when to use each part.

📐 FORMULAS

Formula 1 — Sample Proportion (Point Estimate)
p-hat = x / n
p-hat= sample proportion (best estimate of population proportion)
x= number of successes / favorable cases in the sample
n= total sample size
WHEN TO USE: Any time you need the "best estimate" of the proportion. Always compute p-hat FIRST before anything else.
Formula 2 — q-hat (Complement)
q-hat = 1 − p-hat
q-hat= proportion of failures (1 minus the successes proportion)
WHEN TO USE: Always compute q-hat right after p-hat. You need it for Standard Error.
Formula 3 — Standard Error (SE)
SE = sqrt( p-hat × q-hat / n )
SE= standard error (how much the sample proportion varies from the true proportion)
p-hat= sample proportion (from Formula 1)
q-hat= 1 − p-hat (from Formula 2)
n= sample size
WHEN TO USE: After computing p-hat and q-hat. Measures the "accuracy" of the estimate.

💡 Worked Examples

Example 1 — n=400, x=250 ▼ Show Solution
n=400 observations, x=250 successes. Find p-hat, q-hat, and SE.
Step 1 — p-hatp-hat = x/n = 250/400 = 0.625 (or 62.5%)
Step 2 — q-hatq-hat = 1 − 0.625 = 0.375
Step 3 — SESE = sqrt(0.625 × 0.375 / 400) = sqrt(0.234375/400) = sqrt(0.000586) = 0.024
Example 2 — Filipino Women Survey (1500, x=810) ▼ Show Solution
1500 Filipino women, 810 conscious of figure, 975 exercise twice a week. Find p-hat and SE for each.
Part A — Conscious of figure p-hat = 810/1500 = 0.540 | q-hat = 1 − 0.540 = 0.46
SE = sqrt(0.540 × 0.46 / 1500) = sqrt(0.2484/1500) = sqrt(0.0001656) = 0.013
Part B — Exercise twice a week p-hat = 975/1500 = 0.650 | q-hat = 1 − 0.650 = 0.35
SE = sqrt(0.650 × 0.35 / 1500) = sqrt(0.2275/1500) = 0.012
Example 3 — 300 Households, x=120 ▼ Show Solution
300 households, 120 not segregating garbage. Find p-hat and SE.
Solution p-hat = 120/300 = 0.400 | q-hat = 1 − 0.400 = 0.600
SE = sqrt(0.400 × 0.600 / 300) = sqrt(0.240/300) = sqrt(0.0008) = 0.028

✏️ Key Terms

Population Proportion (p)True proportion of the whole population with a given attribute (parameter)
Sample Proportion (p-hat)Proportion from a sample — used as the BEST ESTIMATE of p
q-hatSample proportion of failures = 1 − p-hat

📊 MODULE 12 — Confidence Interval for Population Proportion COMPUTATION

Focus on COMPUTATIONS. Know the formula structure and how to plug in values.

📐 FORMULA — Confidence Interval for Proportion p

Confidence Interval Formula
p-hat − Z(α/2)·sqrt(pq/n) < p < p-hat + Z(α/2)·sqrt(pq/n)
Lower Limit = p-hat − Z(α/2) × sqrt(p-hat × q-hat / n)
Upper Limit = p-hat + Z(α/2) × sqrt(p-hat × q-hat / n)
p-hat= sample proportion = x/n
q-hat= 1 − p-hat
Z(α/2)= critical value from z-table based on confidence level
n= sample size
p= true population proportion (what we are estimating)

🔑 Critical Values — Commonly Used Confidence Levels MEMORIZE

Confidence Level (1−α)Alpha (α)Critical Value Z(α/2)
90%0.101.645
95%0.051.96
99%0.012.575
🧠 MNEMONIC: "90=1.645 | 95=1.96 | 99=2.575" → Count the confidence up, count the critical up!

💡 Worked Example

Example — n=400, x=250, Find 95% CI ▼ Show Solution
n=400, x=250. Find 95% confidence interval for population proportion p.
Step 1 — Find p-hat and q-hatp-hat = 250/400 = 0.625 | q-hat = 1 − 0.625 = 0.375
Step 2 — Identify Z(α/2)95% CI → Z(α/2) = 1.96
Step 3 — Compute the margin sqrt(pq/n) = sqrt(0.625 × 0.375 / 400) = sqrt(0.000586) = 0.0242
Margin = 1.96 × 0.0242 = 0.0474
Step 4 — Compute CI Lower = 0.625 − 0.0474 = 0.578
Upper = 0.625 + 0.0474 = 0.672
Answer: 0.578 < p < 0.672
Step 5 — InterpretWe are 95% confident that the population proportion will be between 0.578 and 0.672.
Example 2 — n=400, x=250, Find 99% CI ▼ Show Solution
Same data: n=400, x=250. Find 99% confidence interval.
Givenp-hat=0.625, q-hat=0.375, Z(α/2) for 99% = 2.575
Compute CI Lower = 0.625 − 2.575 × sqrt(0.625×0.375/400) = 0.625 − 0.0623 = 0.563
Upper = 0.625 + 0.0623 = 0.688
Answer: 0.563 < p < 0.688
Notice: The 99% CI is WIDER than the 95% CI. Higher confidence = wider interval!

🔍 Interpretation Template

We are [confidence level]% confident that the [population being described] is between [lower limit] and [upper limit].

📊 MODULE 13 — Sample Size Determination COMPUTATION

Focus on COMPUTATIONS. Key: always round UP to the next whole number!

📐 TWO FORMULAS — Know Which One to Use!

Formula A — Sample Size for Estimating Population MEAN (µ)
n = ( Z(α/2) × σ / e )²
n= required sample size (always round UP)
Z(α/2)= critical value (1.645 for 90%, 1.96 for 95%, 2.575 for 99%)
σ= population standard deviation (given in problem)
e= margin of error (maximum allowable error)
USE WHEN: Problem asks about estimating the MEAN (µ). Sigma (σ) and error (e) are given.
Formula B — Sample Size for Estimating Population PROPORTION (p)
n = Z(α/2)² / ( 4e² )
n= required sample size (always round UP)
Z(α/2)= critical value based on confidence level
e= margin of error
4e²= 4 × e × e (denominator)
USE WHEN: Problem asks about estimating a PROPORTION (p). No sigma given — only confidence level and margin of error.
🚨 IMPORTANT RULE: ALWAYS round up to the next whole number! e.g., 42.3 → 43, never 42.

🔍 How to Tell Which Formula to Use

If the problem mentions...Use Formula
"standard deviation" (σ), "mean" (µ)Formula A — for estimating µ
"proportion", "percentage", "%", no σ givenFormula B — for estimating p

💡 Worked Examples

Example 1 — Formula A: 90% confidence, e=0.5, σ=2 ▼ Show
How large a sample should be taken to be 90% sure that the error does not exceed 0.5 when σ=2?
IdentifyEstimating MEAN → Use Formula A. 90% CI → Z=1.645, σ=2, e=0.5
Computen = (1.645 × 2 / 0.5)² = (3.29 / 0.5)² = (6.58)² = 43.3 → Round up: n = 43
Example 2 — Formula B: 99% confidence, e=0.02 ▼ Show
How large a sample should be taken to be 99% confident that our estimate of p is within 0.02?
IdentifyEstimating PROPORTION → Use Formula B. 99% CI → Z=2.575, e=0.02
Computen = (2.575)² / (4 × 0.02²) = 6.6306 / (4 × 0.0004) = 6.6306 / 0.0016 = 4144 → n = 4144

📗 MODULE 16 — Null & Alternative Hypothesis, Types of Error, Rejection Region CONCEPT

Focus on UNDERSTANDING the concepts: what H0 and Ha mean, types of error, and test directions.

✏️ Core Definitions

TermDefinition
HypothesisA claim, statement, or tentative explanation about people, objects, or events
Statistical HypothesisAssumption about the distribution of a population — either true or false
Hypothesis TestingStatistical procedure to determine which hypothesis is more acceptable
Null Hypothesis (H0)States NO difference or relationship exists. Always uses =, ≥, ≤
Alternative Hypothesis (Ha)States a difference DOES exist. Opposite of H0. Uses ≠, <, >
Level of Significance (α)Probability of committing Type I error. Common: 0.01, 0.05, 0.10
Rejection RegionSet of values where H0 is REJECTED (the "tail area")
Critical ValueBoundary point between rejection and non-rejection regions

📗 MODULE 17 — Basic Steps of Hypothesis Testing & Types of Test Statistic CONCEPT

🪜 The 5 Steps in Hypothesis Testing ⭐ EXAM LIKELY

🧠 MNEMONIC: "F-I-D-D-S" → Formulate, Identify, Determine, Decide, State
1
FORMULATE the Null and Alternative Hypothesis — Write H0 (no difference, uses =) and Ha (difference exists, uses ≠ < >)
2
IDENTIFY the Level of Significance (α) and Test Statistic — Choose z-test, t-test, or F-test. α = 0.01, 0.05, or 0.10
3
DETERMINE the Critical Value and Rejection Region — Use α for one-tailed; α/2 for two-tailed. Look up in z or t table
4
DECIDE — Compute the Test Statistic and Compare — If |computed| > |critical| → REJECT H0
5
STATE the Conclusion — "At the ___ level of significance, there is [enough/not enough] evidence to [support/reject] the claim that ___."

📐 Formulas Side by Side

Z-TEST (σ known OR n ≥ 30)
z = (x̄ − µ) / (σ / √n)

T-TEST (σ unknown AND n < 30)
t = (x̄ − µ) / (s / √n)
s = sample SD. Also need: df = n − 1 for t-table lookup.

📗 MODULE 22 — Analysis of Variance (ANOVA) CONCEPT COMPUTATION

Focus on CONCEPTS and understanding the formulas. The F-ratio is the test statistic for ANOVA.

✏️ What is ANOVA?

ANOVA is a comparison test used to determine the significant difference among normal population means. Used when comparing means of 3 or more populations.

Assumptions for ANOVA:

  • Groups are from normal populations
  • Variances of the different groups are equal
  • Random samples in groups are independent
The test statistic for ANOVA is the F-ratio. Compare computed F to tabular F value.

📐 ANOVA Formulas — Step by Step

1. Total Sum of Squares (TSS)
TSS = Σ(x²) − (Σx)²/N
Σ(x²)= sum of all squared values (every x² added up)
(Σx)²= square of the sum of ALL x values
N= total number of all data points (rows × columns)
2. Sum of Squares Between Columns (SSb)
SSb = (1/rows) × Σ(column sum)² − (Σx)²/N
rows= number of items per group (each group has equal rows)
Σ(column sum)²= square each group's total, then add them
3. Sum of Squares Within Columns (SSw)
SSw = TSS − SSb
4. Mean SS Between (MSSb)
MSSb = SSb / dfb     where dfb = columns − 1
5. Mean SS Within (MSSw)
MSSw = SSw / dfw     where dfw = (rows × columns) − columns
6. F-value (Test Statistic)
F = MSSb / MSSw
DECISION: If computed F < tabular F → ACCEPT H0 (no significant difference). If computed F > tabular F → REJECT H0.

💡 Worked Example — Teaching Methods

ANOVA — 3 Groups, 7 Students Each (5% level) ▼ Show Full Solution
Group A (Exploratory): 85,90,92,88,91,93,89 | Group B (Cooperative): 86,88,89,90,87,88,91 | Group C (Traditional): 100,89,88,87,83,85,80
Step 1 — HypothesesH0: No significant difference among teaching methods. | Ha: There IS a significant difference.
Step 2α=0.05, test statistic = ANOVA (F-test)
Step 3 — Critical Fdfb=3−1=2 (columns), look up F-table: col 2, row 18 → F-critical = 3.55
TSSΣ(x²) = 56384+54755+53748 = 164,887 | Σx = 628+619+612 = 1859
TSS = 164,887 − (1859)²/21 = 164,887 − 164,566 = 321
SSb= (1/7)[(628)²+(619)²+(612)²] − (1859)²/21
= (1/7)[394384+383161+374544] − 164566
= (1/7)(1152089) − 164566 = 164584.14 − 164566 = 18.38
SSw= TSS − SSb = 321 − 18.38 = 302.86
MSSb= 18.38/2 = 9.19
MSSwdfw = (7×3)−3 = 18 | MSSw = 302.86/18 = 16.825
F-valueF = 9.19/16.825 = 0.546
Decision0.546 < 3.55 → ACCEPT H0
ConclusionThe three strategies of teaching are NOT significantly different from each other at 5% level of significance.

📊 MODULE 23 — Pearson's Correlation Coefficient (r) COMPUTATION MEMORIZE Interpretations!

⚠️ Teacher said: "You are REQUIRED to MEMORIZE the interpretation of the different values of Pearson's r. It is NOT included in the formulas that will be provided."

📐 FORMULA — Pearson's r

Pearson Correlation Coefficient
r = [ n·Σxy − Σx·Σy ] / sqrt{ [n·Σx² − (Σx)²] × [n·Σy² − (Σy)²] }
r= Pearson correlation coefficient (ranges from −1 to +1)
n= number of paired data points
Σxy= sum of (each x × each y) products
Σx= sum of all x values
Σy= sum of all y values
Σx²= sum of each x squared
Σy²= sum of each y squared
(Σx)²= square of the SUM of x (different from Σx²!)
Setup tip: Create a table with columns: x, y, x², y², xy. Get sums of each column, then plug into formula.
